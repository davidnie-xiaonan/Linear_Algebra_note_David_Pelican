Online course on basic linear algebra

This is an online course note on basic linear algebra. The reason why this note exists is because of the COVID-19 pandemic. There is absolutely nothing to do so writing this note became the only fun during the time.

The goal is to give a broad introduction to linear algebra without much proof or calculations. For more introduction, got to About this note.

Currently, this page is still under construction. The problem sets for each chapter is incomplete. Feel free to point out any error and the contact information can be find on my Linkedin.
